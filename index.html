Learn about Decimal to Binary and Binary to Decimal Conversion

Decimal to Binary Conversion

Converting a decimal number to binary involves repeatedly dividing the decimal number by 2 and recording the remainders. The binary representation is obtained by reading the remainders in reverse order.

Binary to Decimal Conversion

Converting a binary number to decimal involves multiplying each binary digit by the corresponding power of 2 and then summing the results. The rightmost bit represents 2^0, the next bit represents 2^1, and so on.

For example, to convert the binary number 1101 to decimal: (1 * 2^3) + (1 * 2^2) + (0 * 2^1) + (1 * 2^0) = 8 + 4 + 0 + 1 = 13.

Addition of Decimal Numbers

In decimal addition, numbers are added column-wise, starting from the rightmost column. If the sum in a column exceeds 9, a carry is generated to the next column on the left.

Addition of Binary Numbers

Binary addition follows the same principles as decimal addition, but with a base of 2. In binary, the sum in each column can be 0, 1, or 10 (when the sum is 2). Carryovers occur when the sum is 10, and they move to the next column on the left.

For example, to add 1101 and 101: 

                       1101
                   +   101
                   __________
                      10010
